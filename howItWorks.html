Easy Grocery

How It Works

Our process for calculating how much the user should be eating overall as well as how the foods should be divided for each food group is based on the user's age and gender.

This calculation is accomplished through the following process:

  1. Take the user's age and gender
  2. Get the average caloric intake based on that info
  3. Percentage of each food group is based on person's age
  4. Distribute the calories between the five food groups
  5. Generate list or items for user and create list
  6. Present checklist for user to use while grocery shopping
